Manage docker dengan mudah menggunakan portainer.io
Created 2019-08-05| Updated 2019-08-05|docker

mulai dari Install, sampai manage docker menggunakan portainer. Membuat docker lebih manusiawi, dan mudah dipahami.

Pembukaan


Hallo semuanya, kembali lagi dengan saya. Dan kali ini saya akan memberikan tutorial mulai dari install sampai memanage docker menggunakan portainer.
Jika kalian baru pertama kali menggunakan docker, dan mencari cara agar docker lebih manusiawi berarti kalian tepat mengunjungi post ini, karena akan dijelaskan agar docker lebih manusiawi .

Awalan

Untuk memanage docker biasanya kita menggunakan command line atau CLI dan bagi sebagian orang pusing, dan tidak bisa mengingat perintah yang harus dijalankan, maka dari itu ada terobosan untuk menggunakan / memanage docker secara GUI, dan portainer ini berbasis web.
Jadi portainer itu sebuah image yang nantinya akan dijalankan sebagai container, untuk remote docker.

Masuk ke tutorial

Oke langsung saja masuk ketutorialnya, sebelum memanage docker secara GUI, download/pull image portainer terlebih dahulu dari docker hub, jika kalian ada kendala kuota santai saja, karena hanya membutuhkan 10MB saja.
Kecil bukan? ya sangat kecil dan ringan pastinya.

Untuk pull image nya bisa menggunakan perintah :

Jika sudah selesai proses pull image nya, lanjut kita jalankan portainer nya menggunakan perintah :

Pada perintah untuk menjalankan container nya dalam mode detach atau running di background, jadi container yang sudah dibuat akan tetap berjalan di background.
portainer menggunakan port 9000, dan lakukan port expose antara host dan container -p 9000:9000.
supaya portainer bisa meremote docker secara local di satu host, harus menggunakan option -v /var/run/docker.sock:/var/run/docker.sock.
dan yang terakhir pilih nama image nya, yaitu portainer/portainer.

Selanjutnya akses ip_host:9000 di browser kesayangan kalian, dan nanti akan muncul tampilan seperti gambar dibawah :

Pada gambar diatas kalian harus mengisi password terlebih dahulu sebagai useradmin.

Selanjutnya pilih kemana portainer ingin dihubungkan, apakah ingin ke local, ataupun remote. Dan pada tutorial kali ini saya akan meremote secara local.
