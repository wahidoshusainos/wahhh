mempercantik tampilan openSUSE leap 42.3
Created 2019-07-29| Updated 2019-07-29|openSUSE

Mempercantik tampilan openSUSE Leap 42.3 dengan mudah, tampilan menjadi fresh dan enak dipandang.

Pembukaan

Hallo semuanya, kembali lagi dengan saya wahid husain di blog yang ingin bermanfaat ini, kali ini saya mau share cara mempercantik tampilan openSUSE Leap 42.3 , mungkin bagi sebagian orang yang ingin migrasi ke linux akan berpikir :

“Banyak aplikasi yang gak bisa dijalanin

“Tampilannya gak bagus, gw gak suka”

Nah pernyataan nomor 2 itu yang kali ini saya tepis ctangggggg anggap aja sound effect Oke lanjut, untuk tampilan di linux itu bisa banget buat di modifikasi, apalagi sekarang udah banyak distro linux yang tampilannya bisa dibilang menyejukkan mata.
Nah bagi yang penasaran gimana tampilan linux yang saya gunakan nih penampakannya :

Bagus kan? Mantap kan?? nih lagi

Udah lah dua aja gambarnya, ohya disini saya pakai openSUSE Leap 42.3 dan desktop environment nya XFCE, dan juga theme nya saya pakai Numix.

Download icon Numix nya

Pertama-tama download icon numix nya, karena disini saya menggunakan openSUSE jadi download aja di SINI , saya pilih yang leap 42.3 ya sesuaikan saja sama yang digunakan kalian

Pilih yang 1 click install biar nanti otomatis ngedownload + install, dan langsung bisa digunakan, kalau sudah di download file nya akan berekstensi ymp , nah file itu klik aja dua kali, nanti langsung download kok.

Apply icon

Selanjutnya tinggal apply deh icon nya, atau terapin.
Untuk nerapinnya bisa di Setting Manager trus pilih Appearance , di tab icon pilih icon numix nya.

Install Theme / Style

Selanjutnya download theme numix nya, untuk download nya bisa melalui terminal, dan perintahnya adalah :

Pastiin ngejalanin perintahnya dalam mode root yak.

Apply Theme / Style

Untuk apply theme nya pun sama seperti nerapin iconnya, hanya bedanya dia di tab style dan pilih deh style nya.

Ganti tampilan window

Setelah udah apply theme + icon, pasti akan ada yang kurang yaitu window nya masih gak enak dipandang, yaudah setting deh window nya, untuk setting ini bisa di Setting manager dan pilih Window manager dan pilih lagi numix biar matching dengan icon + theme nya.

Intinya jadi makin betah buat singgah di openSUSE, jangan pindah loh udah stay aja , kan udah mantep tampilannya huehe.
