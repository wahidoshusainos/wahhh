Instalasi nginx docker
Created 2019-07-29| Updated 2019-07-29|docker

nginx merupakan web server yang dapat diinstall di beberapa OS linux, tetapi disini saya akan memberi tutorial bagaimana membangun nya menggunakan Docker, dan tentu saja OS yang saya gunakan adalah openSUSE Leap 42.3, berikut tutorialnya.

Pembukaan

OKe mulai yak, siap dah pembukaan dulu.. Haii semuanya kali ini saya akan memberikan tutorial yang sangat bermanfaat untuk kalian ataupun orang lain, jadi jangan sungkan ataupun malu untuk membagikan tutorial ini, dan tutorial yang akan dibawakan pada kesempatan kali ini adalah Cara install nginx di docker , nah itu judul dari tutorial kali ini.
Dan saya menggunakan openSUSE Leap 42.3 sebagai host OS nya.

Sedikit penjelasan

OKe sebelum masuk ketutorial nya, sedikit penjelasan saja ya mengenai nginx nya.

Nginx (baca: engine x) adalah server HTTP dan Proxy dengan kode sumber terbuka yang bisa juga berfungsi sebagai proxy IMAP/POP3. â€¦ Nginx terkenal karena stabil, memiliki tingkat performansi tinggi dan minim mengonsumsi sumber daya.

Nah penjelasannya diatas dan penjelasan diatas saya ambil dari wikipedia seharusnya kalian juga sudah mengetahui apa itu nginx ya tapi gak papa lah di beri tahu lagi disini.
Nginx itu biasanya diinstal di suatu OS, nah gimana jadinya kalau nginx diinstal di Docker, jawabannya bisa dong karena pada docker hub pun sudah menyediakan image nginx dan itu official jadi kita bisa menjalankan nginx di docker, bisa jadi semakin ringan deh.
